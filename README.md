# Spring Boot User List Demo

MY FIRST Web of Springboot :)

This is a simple Spring Boot web project that reads user data from a local text file and displays the user list on a web page.



## Technologies Used

- Java
- Spring Boot
- Spring Web
- Lombok
- HTML
- JavaScript
